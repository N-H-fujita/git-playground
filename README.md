# git-playground

Git 操作を実際に試しながら学ぶための検証用リポジトリです。

## 目的

- Git の基本操作を「履歴込み」で理解する
- amend / reset / stash / force push など、
  事故りやすい操作を安全な場所で試す
- 「なぜそうしたか」を言語化する練習

## 方針

- 失敗を前提にする
- 履歴を書き換える操作も恐れず行う
- 各実験は README にメモを残す
- commit log も教材の一部と考える

## メモ

このリポジトリは成果物ではなく、
**学習の過程そのもの**を残すことを目的とする。

### 01. commit --amend
- 目的：commit メッセージを修正する
- やったこと：README を編集
- 失敗：
  - commit メッセージを適当に書いた
- 学んだこと：
  - push 前なら commit --amend で安全に修正できる
  - amend は新しい commit を作るのではなく、直前の commit を書き換える
